# Xuwer

Xuwer is a modern SaaS landing page concept for a business discovery platform. The idea behind Xuwer is to help people discover businesses based on their interests, preferences, and goals instead of scrolling through endless search results.

I built this project to improve my frontend development skills by focusing on responsive design, modern UI/UX principles, animations, and creating a clean SaaS-style user experience.

## 🚀 Live Demo

🔗 **Live Website:** https://xuwer03-11sb.netlify.app/

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern dark-themed SaaS interface
- Interactive product preview dashboard
- Smooth GSAP animations
- Sticky navigation bar
- Business discovery UI
- About section and feature highlights
- FAQ section
- Optimized and organized project structure

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6)
- GSAP
- Google Fonts (Inter)
